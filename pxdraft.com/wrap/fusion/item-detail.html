Moxom - Multipurpose Responsive Template

Moxom is a HTML5 template based on Sass and Bootstrap 4 with modern and creative multipurpose design you can use it as a startups, web studio and creative agencies, musicians, architects, etc. There are included five unique homepages page variations. This design is very easy to customize and all files and code has been well organized and nicely commented for easy to customize.

Features

  • 5 Home Page Options
  • about us, contact us, services, portfolio and more inner pages.
  • Working Contact Form
  • Created with Sass
  • Customizable via CSS or SCSS
  • Google Fonts
  • 100% Fluid Responsive
  • Tested on real devices
  • Flexible Layout
  • Flat, modern and clean design
  • Retina Optimized
  • Clean & Commented Code
  • Advanced Typography
  • Built on Bootstrap 4
  • Valid HTML5 Files
  • Font Awesome
  • Themify Icons
  • Awesome Unique Look
  • Quick, Dedicated & Professional Support
  • Well Documented & More …
